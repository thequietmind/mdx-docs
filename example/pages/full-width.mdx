# Full-Width Page

<Lead>
  This page sets `sidebar: false` and `maxWidth: false` in its frontmatter, so
  the docs sidebar is hidden and the content spans the full width instead of the
  default 960px cap.
</Lead>

Use this for landing pages, privacy policies, changelogs, or any page that does
not need the documentation navigation. The site title in the top bar still links
back to the home page.
