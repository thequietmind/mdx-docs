## Patterns

### [Design System](/design-system)

Document your color tokens straight from your live theme — the swatches update
automatically when the theme changes, and the source is included to copy.

### [Typography](/typography)

Document your type scale from `theme.typography`, with the component source to
build the same page yourself.

### [Documentation Components](/components)

Reusable building blocks — lead text, banners, hero sections, and feature grids —
each shown with a live preview and copy-paste source.

### [Embedding React in MDX](/embedding-react)

Drop any React or Material UI component straight into your Markdown — the
foundation everything else here builds on.
